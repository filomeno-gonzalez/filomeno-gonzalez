# Filomeno Gonzalez

**Databricks Resident Solutions Architect** | Dallas, TX

I partner with enterprise engineering and data teams to design, build, and operationalize production-grade data and AI platforms on Databricks. My work spans the full stack — from ingestion architecture and governance to LLMOps and agentic AI — across Azure, AWS, and GCP.

---

## What I Build

**Data Engineering & Pipelines**
Designing medallion architectures and building production Delta Live Tables pipelines for complex enterprise workflows — including tax declaration automation, financial reporting, supply chain optimization, and real-time data migration from legacy ETL tools like Informatica PowerCenter.

**Governance & Identity**
Implementing Unity Catalog from scratch and automating identity lifecycle management at scale using SCIM, Azure Entra ID, and Databricks APIs — eliminating manual provisioning across large multi-workspace environments.

**AI & LLMOps**
Building end-to-end LLM pipelines on Databricks: RAG systems with Azure OpenAI, GraphRAG for connected data, multimodal retrieval, and MLflow-tracked model serving. Focused on making AI production-ready, not just prototype-ready.

**DevOps for Databricks**
CI/CD pipelines using Databricks Asset Bundles, GitHub Actions, and Terraform — enabling teams to ship data products with the same discipline as software engineering.
